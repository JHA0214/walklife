# 워크라이프
고령자가 안전하게 낙상 예방운동을 시작하고, 보호자가 부모님의 운동을 쉽게 도울 수 있도록 하는 노인 맞춤 운동 가이드 앱
주 고객은 시니어로 만 65세 이상이다.
사용자가 혼자 어플 사용시에도 무리없이 운동을 할 수 있을만큼 사용이 편해야한다.

## 기능
검색기능을 통해 걷기, 앉기 등의 동작을 검색하여 운동을 찾을 수 있고, 강화를 원하는 부위를 검색하여 운동을 찾을 수 있다.
검색어를 입력하면 홈 화면 목록이 즉시 필터링되며, 검색 버튼(Enter)을 누르면 별도의 검색 결과 화면으로 이동한다.
검색 단락 밑에 버튼으로 원하는 부위를 고를 수 있게 한다. 이때 상체와 하체 버튼을 만들고 선택시 더 세부적인 부위를 고를 수 있는 페이지가 나온다.
- 상체: 목, 어깨, 팔, 등, 허리
- 하체: 엉덩이, 허벅지, 무릎, 종아리, 발목

운동 가이드 페이지는 관리자가 올린 유튜브 영상을 연결해 상단에는 운동 가이드 영상과 밑으로 간단한 설명을 텍스트로 적는다.
영상은 유튜브 페이지로 이동을 하지 않고 현재 페이지에서 바로 재생이 가능하다.
각 운동에 해시태그를 설정하여 키워드로 검색이 가능하다.
글자 크기 변경과 고대비 켜기 기능과 같은 설정 기능은 설정 페이지에서 관리한다.

### 운동 난이도
각 운동에는 난이도가 별점(★)으로 표시된다. 별 1개(매우 쉬움)부터 별 5개(매우 어려움)까지 0.5개 단위로 세분화되어 있으며, 운동 이름 바로 아래에 표시된다.
난이도는 관리자 페이지에서 운동별로 직접 설정할 수 있다.

### 즐겨찾기
자주 하는 운동은 별표(☆/★) 버튼으로 즐겨찾기에 추가·해제할 수 있다.
홈 화면 상단에 즐겨찾기한 운동만 모아 보여주는 "즐겨찾기" 영역이 있으며, 즐겨찾기가 없으면 표시되지 않는다.

### 음성 카운트
운동 가이드 페이지에서 "운동 시작" 버튼을 누르면 "운동을 시작합니다" 안내와 3-2-1 카운트다운 후, 정해진 간격(기본 2초)마다 "하나, 둘, 셋…"처럼 음성으로 횟수를 세어준다(기본 12회, 완료 시 안내 음성 포함).
손을 쓰기 어려운 상황에서도 소리만으로 운동 횟수를 따라갈 수 있도록 하기 위함이며, 운동별 음성 카운트 간격(초)과 횟수는 관리자 페이지에서 설정할 수 있다.

 ### 관리자 페이지
 첫 화면에서 관리자 로그인 버튼을 작게 추가하여 비밀번호 입력시 관리자 페이지로 넘어간다.
 관리자 페이지에서는 각 운동에 대한 설명 페이지를 생성 및 수정할 수 있고 운동에 대한 영상을 유튜브 링크로 변경할 수 있다.
 관리자 비밀번호는 '007181' 이다.
 각 운동을 수정하거나 생성시 해시태그를 추가할 수 있다.
 각 운동을 수정하거나 생성시 난이도(1.0~5.0, 0.5 단위)와 음성 카운트 간격·횟수도 함께 설정할 수 있다.
 운동을 삭제할 수 있으며, 삭제 시 해당 운동은 즐겨찾기 목록에서도 함께 제거된다.

## 저장 방식
별도의 서버나 데이터베이스 없이, 운동 데이터·설정·즐겨찾기는 브라우저의 localStorage에, 관리자 로그인 상태는 sessionStorage에 저장된다.
따라서 브라우저(기기)를 바꾸면 데이터가 공유되지 않으며, 브라우저 저장소를 지우면 기본 운동 데이터로 초기화된다.

## 기술 구조
프레임워크나 빌드 도구 없이 순수 HTML/CSS/JavaScript(ES 모듈)로 작성되었다.
`js/main.js`가 진입점이며, 화면(홈/부위/검색/운동가이드/설정/관리자)별로 모듈이 나뉘어 있다.
GitHub Pages로 배포하며, 로컬에서 확인할 때는 브라우저 모듈 정책상 `file://`로 직접 열지 않고 로컬 서버(예: `python3 -m http.server`)로 열어야 한다.
